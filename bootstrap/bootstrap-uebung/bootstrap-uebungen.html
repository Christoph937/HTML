📚 Was ist Bootstrap?

Bootstrap ist das beliebteste CSS-Framework der Welt! Es bietet:

  • 🎨 Vorgefertigte Komponenten (Buttons, Cards, Navigation, etc.)
  • 📱 Responsive Grid-System (automatisch mobile-freundlich)
  • ⚡ Schnelle Entwicklung (weniger CSS schreiben)
  • 🎯 Konsistentes Design (professioneller Look)
📖 Offizielle Dokumentation: Bootstrap 5.3 Docs

Aufgabe 1: Bootstrap 5.3 einbinden

Ziel: Bootstrap CSS und JavaScript in deine HTML-Seite einbinden

📖 Ressourcen: Bootstrap Getting Started | W3Schools Bootstrap Start
💡 Konzept:

CDN (Content Delivery Network): Bootstrap wird von einem externen Server geladen - du musst es nicht herunterladen!

CSS im <head>: Für das Styling der Komponenten

JS vor </body>: Für interaktive Komponenten (Dropdown, Modal, etc.)

<!-- Bootstrap CSS im <head> --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <!-- Bootstrap JS vor </body> --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Aufgabe 2: Container und Grid-System verstehen

Ziel: Lerne das Bootstrap Grid-System mit Container, Rows und Columns

📖 Ressourcen: Bootstrap Grid System | Bootstrap Container
💡 Konzept:

Container: Wrapper für dein Content (.container oder .container-fluid)

Row: Horizontale Zeile (.row)

Column: Spalte (.col, .col-6, .col-md-4, etc.)

12-Spalten-System: Eine Row hat maximal 12 Spalten

<div class="container"> <div class="row"> <div class="col-6">Halbe Breite</div> <div class="col-6">Halbe Breite</div> </div> </div>
🎯 Deine Aufgabe: Erstelle das Grid

Aufgabe 3: Navigation mit Logo erstellen

Ziel: Erstelle eine responsive Navbar für unsere Webseite

📖 Ressourcen: Bootstrap Navbar | W3Schools Navbar
💡 Konzept:

Navbar-Klassen: .navbar, .navbar-expand-lg, .navbar-light/.navbar-dark

Navbar-Brand: Für Logo/Markenname (.navbar-brand)

Navbar-Nav: Für Navigation-Links (.navbar-nav, .nav-item, .nav-link)

Responsive: Automatischer Hamburger-Menü auf mobilen Geräten

<nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container"> <a class="navbar-brand" href="#">MeinLogo</a> <div class="navbar-nav"> <a class="nav-link" href="#">Home</a> </div> </div> </nav>

Aufgabe 5: Produkt-Cards in Spalten

Ziel: Erstelle responsive Produkt-Cards mit Bootstrap's Card-Component

📖 Ressourcen: Bootstrap Cards | W3Schools Cards
💡 Konzept:

Card-Struktur: .card → .card-img-top → .card-body → .card-title/.card-text

📋 Bootstrap 5.3 Cheat Sheet

🎨 Layout & Grid
.container / .container-fluid .row .col / .col-6 / .col-md-4 .d-flex / .justify-content-center .text-center / .text-start / .text-end
🎯 Komponenten
.btn / .btn-primary / .btn-lg .card / .card-body / .card-title .navbar / .navbar-brand / .nav-link .form-control / .form-label .modal / .accordion / .carousel
📏 Spacing (Margin & Padding)
.m-3 / .mt-3 / .mb-3 / .mx-3 / .my-3 .p-3 / .pt-3 / .pb-3 / .px-3 / .py-3 (0-5 für verschiedene Größen)
🌈 Farben & Text
.bg-primary / .bg-success / .bg-warning .text-primary / .text-muted / .text-white .fw-bold / .fw-normal .display-1 bis .display-6 / .lead
💡 Pro-Tipps:
  • 📱 Mobile First: Bootstrap ist standardmäßig responsive!
  • 🎨 Customize: Verwende CSS-Variablen für eigene Farben
  • ⚡ Performance: Lade nur die Bootstrap-Komponenten, die du brauchst
  • 🔧 Debug: Nutze Browser-Entwicklertools zum Experimentieren
